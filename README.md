# oh-my-claude-board

> Claude Code orchestration TUI dashboard

Real-time visualization of Claude Code orchestration progress in your terminal.

## Features

- Watch mode: Real-time TASKS.md + Hook event monitoring
- Gantt chart: Phase/Task tree with progress bars
- Error analysis: Rule-based error pattern matching

## Installation

```bash
cargo install oh-my-claude-board
```

## Usage

```bash
oh-my-claude-board watch --tasks ./TASKS.md
```
